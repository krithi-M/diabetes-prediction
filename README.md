# Diabetes Prediction using Machine Learning

## Overview
This project predicts whether a patient has diabetes using Machine Learning techniques.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Dataset
The project uses the Pima Indians Diabetes Dataset.

## Features
- Data Cleaning
- Data Visualization
- Feature Scaling
- Model Training
- Prediction

## Files
- Diabetes Prediction.ipynb
- diabetes.csv
- requirements.txt

## Results
The model was trained and evaluated to predict diabetes based on patient health metrics.
